Token Login

This is a demonstration APP developed using ASP.NET Web API 2, OWIN middleware, ASP.NET Identity and AutoFac to demonstrate the capabilities of C# with token access.

As a developer when wanting to start a fresh project, it is not always easy to create a working skeleton containing this technologies. While there are different projects that provide some of the features, there wasn't a single project that encapsulates everything in order to provide a user with the option to start working right away.

I recommend reading first about the different concepts used in this project (Repository Pattern, Unit of Work Pattern, OWIN, ASP.NET Identity, Access-Token, Dependency Injection) before starting to use it.
